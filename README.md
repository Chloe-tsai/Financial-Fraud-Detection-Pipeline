# 📊 公司財務異常預警系統  
(Financial Fraud Detection Pipeline)

建立一套自動化資料管線，從公開資料來源取得上市櫃公司財報數據，透過資料建模與異常偵測，提供企業風險預警機制。

---

## 📌 前言

隨著企業規模擴大與交易複雜化，財務異常與舞弊風險逐漸提高。  
傳統人工稽核方式不僅耗時，亦難以及時發現潛在問題。

因此，本專題以「上市櫃公司財務異常預警」為核心，  
結合資料工程與分析方法，建立自動化監控與預警系統，  
以提升財務透明度與風險控管能力。

---

## 📊 分析主題  
### 上市櫃公司財務異常偵測

**分析內容：**

- 📈 營收異常變動  
- 💸 成本與費用異常  
- 📉 利潤率異常波動  
- 📊 財務比率異常（ROE、毛利率）  
- 🔁 連續期間異常趨勢  

---

## 🎯 研究目的

- 建立上市櫃公司財務資料整合平台  
- 分析企業財務數據中的異常變化  
- 建立異常偵測規則（Rule-Based）  
- 提供自動化預警機制  
- 協助投資人與企業降低風險  

---

## 📂 資料來源

### 📁 公開資料
- 公開資訊觀測站（MOPS）  
- 政府資料開放平台  
- 上市櫃公司財報資料（營收 / 損益）  

### 📊 輔助資料
- 產業別資料  
- 景氣指標（經濟環境）  

---

## ⚙️ 技術架構

- 🐍 資料收集：Python（爬蟲 / API）  
- 🧹 資料處理：pandas  
- 🗄️ 資料庫：MySQL  
- ⚡ API：FastAPI  
- 📊 視覺化：Streamlit / Redash  
- ⏱️ 排程工具：Airflow
